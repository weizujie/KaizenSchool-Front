# KaizenSchool-Front

> 该项目为 KaizenSchool 在线学习平台的前端部分。


## 使用方法

1. 将该仓库克隆到本地
  ```shell
  git clone https://github.com/weizujie/KaizenSchool-Front.git
  ```
2. 安装依赖
  ```shell
  npm install
  ```
3. 运行该项目
  ```shell
  npm run dev
  ```
